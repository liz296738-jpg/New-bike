## 写在前面

2026年已经过半，AI工具的数量从"每天一个新"进化到了"每小时一个"。打开任何科技媒体，都在告诉你"这个AI神器改变了一切"。但当你真正想找一款解决具体问题的工具时，面对上千款产品，反而无从下手。

这篇文章不推荐具体产品，而是给你一个**分类框架**——有了框架，无论冒出多少新工具，你都知道它属于哪一类、能做什么、和谁竞争。

## 第一层：基础模型层

这是整个AI生态的"地基"。所有上层应用都必须调用它们的能力。

### 大语言模型（LLM）

| 代表产品 | 特点 |
|----------|------|
| Claude（Anthropic） | 长上下文、编程、深度推理最强 |
| GPT-5（OpenAI） | 多模态全能、生态最广 |
| Gemini 3（Google） | 原生多模态、搜索整合深 |
| DeepSeek-V3 | 开源标杆、中文能力强、性价比极高 |
| Qwen 3（阿里） | 开源、中文场景覆盖广 |

### 图像生成模型

- **Midjourney V7** — 审美天花板，设计师首选
- **DALL-E 4** — 与 ChatGPT 深度集成
- **Stable Diffusion 3.5** — 开源可控，适合定制工作流
- **Flux.2** — 文字渲染强，适合海报设计
- **即梦 / 可灵** — 中文 prompt 友好，国内可用

### 视频生成模型

- **Sora 2（OpenAI）** — 物理世界模拟最强
- **可灵 2.0（快手）** — 中文场景、电商短视频
- **Runway Gen-4** — 专业影视级
- **Pika 2.0** — 轻量、快速、适合社交媒体

### 音乐 & 语音

- **Suno V5** — 音乐生成，作词作曲一条龙
- **ElevenLabs** — 语音合成和克隆的行业标准
- **Fish Audio** — 开源中文语音方案

**这一层的要点**：普通用户不需要直接跟模型打交道，但了解各家模型的优劣，能帮你判断上层工具的"内核"够不够强。

## 第二层：开发工具 & 平台层

这一层是给开发者和技术团队用的，在模型之上封装了可调用的能力。

### AI 编程助手

| 类型 | 代表产品 | 一句话 |
|------|----------|--------|
| IDE 内嵌 | GitHub Copilot、Cursor、Windsurf | 在你写代码时实时补全和建议 |
| CLI 智能体 | Claude Code、Codex CLI、Gemini CLI | 能独立完成多步编程任务 |
| 代码审查 | CodeRabbit、Graphite | 自动 Review PR，发现 bug |

### 模型 API & 推理平台

- **Anthropic API / OpenAI API / Google AI Studio** — 直接调用最强模型
- **Together AI / Fireworks / Groq** — 开源模型托管、推理速度极快
- **Replicate** — 一键部署开源模型
- **硅基流动 / 百炼（阿里）** — 国内推理平台

### Agent 开发框架

- **LangChain / LangGraph** — 老牌 Agent 框架，生态最全
- **MCP（Anthropic）** — 模型与工具的标准化协议，2026 年已成行业标准
- **A2A（Google）** — Agent 间通信协议
- **CrewAI / AutoGen / Dify** — 多 Agent 协作、低代码搭建

### 低代码 / 无代码 AI 平台

- **Coze（字节）** — 拖拽式 Bot 搭建
- **Dify** — 开源 LLM 应用开发平台
- **扣子 / 文心智能体** — 国内低代码 Agent

**这一层的要点**：如果你不只是"用"AI，而是"用 AI 来构建东西"，这一层就是你的工具箱。

## 第三层：通用效率工具

这一层是普通用户日常接触最多的 AI 产品，解决通用的办公和创作需求。

### 搜索 & 知识获取

- **ChatGPT Search / Perplexity / 秘塔** — AI 搜索引擎，直接给答案而非链接
- **NotebookLM（Google）** — 上传文档生成播客式摘要，2026 年现象级产品
- **Claude Deep Research** — 多源调研，自动生成引用报告

### 办公套件

- **Microsoft 365 Copilot** — Word/Excel/PPT/Teams 全面 AI 化
- **Google Workspace Gemini** — Gmail/Docs/Sheets 的 AI 伴侣
- **Notion AI** — 笔记、文档、项目管理的 AI 增强
- **飞书智能伙伴 / 钉钉 AI** — 国内办公场景

### 写作 & 内容创作

- **Jasper / Copy.ai** — 营销文案
- **Grammarly** — 英文写作校对
- **通义千问 / Kimi** — 中文长文写作和文档分析

### 设计 & 创意

- **Canva AI** — 模板 + AI 生成，小白也能做设计
- **Figma AI** — UI 设计中的 AI 辅助
- **Photoshop Generative Fill** — 图片局部 AI 重绘

### 沟通 & 会议

- **Otter.ai / Fireflies** — 会议自动转录和总结
- **Slack AI** — 频道消息智能摘要
- **飞书妙记** — 国内会议纪要

**这一层的要点**：如果你的需求是"让我干活更快"，先在这一层找。不需要懂技术，开箱即用。

## 第四层：垂直行业应用

这一层是 AI 在特定行业和场景中的深度应用，往往比通用工具更懂你的业务。

### 销售 & 客服

- **Salesforce Einstein GPT** — CRM 中的 AI 销售助手
- **Intercom Fin** — AI 客服机器人，直接回答客户问题
- **微盟 / 有赞 AI** — 国内电商客服和营销

### 医疗

- **Google Med-PaLM** — 医学知识问答
- **Butterfly Network** — AI 超声影像分析
- **医渡云 / 左手医生** — 国内 AI 辅诊

### 法律

- **Harvey AI** — 律所级法律研究和合同分析
- **CoCounsel（Thomson Reuters）** — 法律文件审查
- **幂律智能** — 国内合同审查

### 金融

- **Bloomberg GPT** — 金融数据和新闻分析
- **Kensho（S&P Global）** — 量化分析
- **蚂蚁集团 AI** — 风控、信贷评估

### 教育

- **Khanmigo（Khan Academy）** — 苏格拉底式 AI 导师
- **Duolingo Max** — AI 角色扮演口语练习
- **学而思 AI / 作业帮** — 国内 AI 辅导

### 工业 & 制造

- **Siemens Industrial Copilot** — 工业自动化编程
- **Covariant** — 仓库机器人 AI 大脑
- **创新奇智** — 国内工业视觉检测

**这一层的要点**：垂直行业工具的特点是"窄而深"——它们只解决一个问题，但解决得比通用工具好得多。

## 怎么用这个框架选工具？

**第一步**：确定你是哪个角色。

- 普通用户 → 直接去**第三层**找通用效率工具
- 开发者 → 关注**第二层**，选编程助手和 API 平台
- 行业从业者 → 看看**第四层**有没有你的行业专用工具

**第二步**：追溯"内核"。

一款 AI 产品吹得再好，核心调用的是哪个模型才决定它的真实上限。如果一个"AI 写作工具"底层用的是三年前的模型，效果不会好到哪里去。

**第三步**：不要追逐"全家桶"。

目前没有哪一款工具能覆盖所有场景。最好的策略是：**选 2-3 款核心工具深度使用，其他按需补充**。

## 最后

AI 工具的分类不是一成不变的。2024 年我们还把"AI 搜索"归入垂直应用，2025 年它就成了通用工具的标准配置。分类框架的价值不在于精确，而在于帮你**建立认知地图**——知道"这个坑位已经有了谁"，以及"哪里还是空白"。
